SVG (Scalable Vector Graphics)

SVG allows for creating vector-based graphics:

Key SVG elements:

  • <rect>: Rectangle
  • <circle>: Circle
  • <ellipse>: Ellipse
  • <line>: Line
  • <polygon>: Multi-sided shape
  • <text>: Text in SVG

SVG Path Element

The path element allows creating more complex shapes:

Path commands:

  • M: Move to
  • L: Line to
  • H: Horizontal line
  • V: Vertical line
  • Q: Quadratic Bézier curve
  • T: Smooth quadratic Bézier
  • Z: Close path
